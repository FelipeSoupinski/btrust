### Ambiente de Desenvolvimento (com Hot-Reload)

1.  **Para iniciar o ambiente:**
    ```bash
    docker-compose up --build
    ```

2.  **Acesse em:** [http://localhost:5173](http://localhost:5173)

As alterações no código serão refletidas em tempo real.

---

### Ambiente de Produção (Build final)

Para construir a imagem de produção (usando Nginx), use o Dockerfile original:
```bash
docker build -t btrust-prod .
docker run -d -p 8080:80 btrust-prod
```
